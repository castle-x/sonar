# Monitor Hub 2.0 预览版 - 重大更新

## 📋 更新日志

### 1. 数据快照/报告功能（核心新功能）
- 报告持久化在后台任务运行，实时显示进度
- 报告展示包含报告详情、总体结论、智能分析、与用例详情数据
- 报告详情支持在线编辑、设置颜色、拖动排序等能力
- 总体结论支持在线编辑富文本，采用现代化文档引擎渲染样式
- 用例详情数据支持标签页切换、预加载模式优化前端体验
- 用例详情数据的展示与实时监控一致，采用汇总表格与图表组两部分

### 2. 首页优化
- 首列数据颜色修改，视觉层次更清晰
- 报告列表支持翻页、高级筛选、高级 SQL 查询
- 点击报告行直接跳转详情页

### 3. 实时监控页面优化
- 合并数据源详情与 Pushgateway 详情，减少冗余信息
- 页面布局更简洁、工整

### 4. 其他功能增强
- 自定义图标上传（数据源、报告均支持）
- 悬浮工具栏：图例切换、布局切换、回到顶部、用例快速切换
- 表格数据一键复制、导出 CSV
- Ctrl+K 全局快速搜索
- 深色/浅色主题切换

## 📝 录制建议

1. **分辨率**: 1920x1080 或 2560x1440
2. **帧率**: 30fps（GIF 可降至 15fps）
3. **格式**: 
   - 单个功能演示 → GIF（便于嵌入文档）
   - 完整演示 → MP4（便于分享）
4. **鼠标高亮**: 建议开启鼠标点击高亮效果
5. **操作节奏**: 每个操作后停留 1-2 秒，让观众看清变化
6. **数据准备**: 
   - 准备 3-5 个有代表性的报告（含多用例）
   - 准备 AI 分析的假数据（Markdown 格式，含表格、代码块）
   - 确保数据源有实时数据在推送
